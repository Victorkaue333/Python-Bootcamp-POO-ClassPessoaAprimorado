# Python Bootcamp - Classe Pessoa Aprimorada

Este repositório faz parte dos meus estudos durante um bootcamp de Python. O objetivo é praticar e aprimorar conceitos de Programação Orientada a Objetos (POO) na linguagem Python.

## Sobre o Projeto

O arquivo `pessoaAprimorada.py` contém uma implementação aprimorada da classe `Pessoa`, com funcionalidades adicionais para demonstrar conceitos como:
- Criação de classes e objetos
- Encapsulamento de atributos
- Métodos de instância e de classe
- Validação de dados
- Princípios de boas práticas em POO

Este código é voltado para fins educacionais e serve como base para fixação dos conceitos aprendidos no bootcamp.

## Como usar

Basta executar o arquivo `pessoaAprimorada.py` para testar as funcionalidades implementadas. Sinta-se à vontade para modificar e experimentar o código!
